### 팀 작업중 발생한 문제

#### develop 작업 브랜치에서의 병합 과정 충돌 
**원인 :** 각기 기능 구현을 같은 파일 내에서 개발, 병합이 이루어짐 

**해결 :** 이러한 경우, 복잡하지만 각 브랜치에 전부 develop의 최신 버전을 다시 pull하여 진행한다 

**방지 :** 각기 기능을 독립성가질 수 있게 서로 영향을 주지 않게 파일 분리
